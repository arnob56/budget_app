# Simple Budget App

## Features
- Add income & expenses
- SQLite database
- Expense pie chart
- Balance over time line chart
- Tkinter GUI
- Logging
- Unit tests

## Setup

python -m venv venv
source venv/bin/activate   (Linux/Mac)
venv\Scripts\activate      (Windows)

pip install -r requirements.txt

## Run App

python -m budget_app.main

## Run Tests

pytest
